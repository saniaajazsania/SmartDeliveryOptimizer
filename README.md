🚀 Smart Delivery Route Optimizer
An AI-based simulation system to solve real-world logistics routing problems using classical AI search algorithms and a Goal-Based Agent.

🧠 Algorithms Implemented
Uninformed Search: BFS, DFS (with max_depth), Uniform Cost Search (UCS)
Informed Search: A* (with configurable heuristic weight), Hill Climbing (with random restarts)
Agent: Goal-Based Agent using Nearest-Neighbor strategy

📊 Features
Interactive Tkinter + Matplotlib GUI
Real-time path animation on a grid map
Runtime parameter modification (Sliders for Heuristic weight, Max Depth, etc.)
10 Pre-defined Test Scenarios (Easy to Full Hard)
Algorithm comparison metrics (Cost, Time, Nodes Explored)

🛠️ Tech Stack
Language: Python
Libraries: NumPy, Matplotlib, Tkinter
